# CLIP 모델 비교: open_clip ViT-B/32 vs cn_clip ViT-B/16

> 작성일: 2026-05-05 | Phase: clip-embedding (Step 2.6)
> 데이터셋: poc_sample 161k 자산 (디자인 에셋 — 로고·포스터·UI 배너)

---

## 측정 환경

| 항목 | 값 |
|------|-----|
| GPU | RTX 4060 8GB |
| 임베딩 수 | 160,036건 × 2모델 = 320,072행 |
| HNSW 인덱스 | m=16, ef_construction=64 |
| API | FastAPI + uvicorn, 127.0.0.1:18000 |

---

## 1. 응답 Latency (logo 쿼리, 60회 반복)

| 모델 | p50 | p95 |
|------|-----|-----|
| clip-vit-b32 (open_clip) | 21.6ms | 34.7ms |
| cn-clip-vitb16 (cn_clip) | 21.5ms | **25.5ms** |

→ p50은 동일. cn_clip의 p95가 9ms 낮아 latency 일관성 우위.

---

## 2. 한국어 텍스트 쿼리 결과 (top-1)

| 쿼리 | open_clip top-1 | cn_clip top-1 | 평가 |
|------|----------------|--------------|------|
| 강아지 | 투명남과 인간녀 포스터 로고 | 로고 단색.png | 둘 다 무관 |
| 그라데이션 배경 | 원형 배너 (색감 일부 유사) | 로고 단색.png | open_clip 소폭 우위 |
| 키즈 캐릭터 | 세로 포스터 | 단편상세 로고 | 둘 다 약함 |
| 타이포그래피 포스터 | 가로형 배너 | 프로모션 배너 | 육안 유사, 동등 |
| 브랜드 로고 | unnamed.jpg | 로고 단색.png | cn_clip 소폭 우위 |
| 썸네일 | 컨테이너메뉴 | 로고 removebg | open_clip 소폭 우위 |
| 버튼 디자인 | 컨테이너메뉴 | 프로모션 로고 | open_clip 소폭 우위 |

**소견**: 한국어 쿼리 모두 의미적으로 정확한 결과를 반환하지 못함. 데이터셋이 자연사진이 아닌 그래픽 디자인 에셋이라 두 모델 모두 domain mismatch 존재. cn_clip이 "브랜드 로고" 등 일부 쿼리에서 로고 이미지를 반환한 것은 의미 일치로 볼 수 있으나, 나머지 쿼리에서 "로고 단색.png"로 수렴하는 패턴(visual anchor bias)이 관찰됨.

## 3. 영문 텍스트 쿼리 결과 (top-1)

| 쿼리 | open_clip top-1 | cn_clip top-1 | 평가 |
|------|----------------|--------------|------|
| blue gradient | 한산 단편상세 로고 | 풀밭 위의 오찬 포스터 | 둘 다 약함 |
| kid character | **키즈 원형 배너** | 야카디3 포스터 | **open_clip 명확 우위** |
| logo | 미션 임파서블 로고 배너 | **방송사로고 이미지** | 동등 (cn_clip 더 직접적) |
| typography poster | 던전흑탑 단편상세 로고 | 마이 아티 필름 포스터 | 동등 |
| thumbnail | 릴로&스티치 로고 배너 | Skylife홈 빅배너 로고 | 둘 다 약함 |

**소견**: "kid character" 에서 open_clip이 키즈 섹션 UI 요소를 정확히 반환. 영문 쿼리에서 open_clip이 전반적으로 시맨틱 이해도 우위.

## 5. 종합 평가

| 기준 | open_clip ViT-B/32 | cn_clip ViT-B/16 |
|------|-------------------|-----------------|
| 한국어 쿼리 | ★★☆ | ★★☆ |
| 영문 쿼리 | **★★★** | ★★☆ |
| 이미지-이미지 유사도 | (독립적) | (독립적) |
| Latency p50 | 21.6ms | 21.5ms |
| Latency p95 | 34.7ms | **25.5ms** |
| 학습 데이터 | LAION-2B (영문 중심) | LAION-CN + WuKong |

**결론**: 두 모델 모두 그래픽 디자인 에셋에 특화된 fine-tuning 없이는 한계 명확. 영문 쿼리 시맨틱 이해는 open_clip, latency 일관성은 cn_clip 소폭 우위. 한국어 쿼리 품질 차이는 이 데이터셋에서 통계적으로 유의미하지 않음.

→ ADR-007 참조.
