## 지금까지 확정된 것

- **아키텍처:** PostgreSQL + pgvector + Python 워커(Celery/Dramatiq) + n8n(오케스트레이션만) + Immich 포크 검토
- **파일 전략:** 원본 재구조화 금지. 하이브리드(논리 레이어 + 점진 이관)
- **PoC 샘플:** 최근 + 자주 찾는 프로젝트 폴더 1개 (구체 폴더 미정)
- **PoC 서버:** 현 PC (WSL2, RTX 4060 8GB, 64GB RAM, 빈 디스크 100GB)
- **격리 전략:** Docker Compose 네임스페이스 `dam`, 비표준 포트 사용
- **모델:** 1차 CLIP ViT-B/32 다국어 버전

## 현재 진행 중

## 다음 액션 (Q3)

### Q3-1. 디스크 확장 방침 결정 ⏳
- (a) 100GB 내에서 20~30GB 샘플로 진행
- (b) 외장 SSD 1TB 추가
- (c) 사내 서버/NAS 일부 할당

### Q3-2. PoC 폴더 선정
- 평균 파일 크기 감
- 100GB 이내 폴더 존재 여부
- 디자이너와 상의 가능 여부

### Q3-3. 현재 PC 상태 점검 명령어 (재시작 후 실행)
```bash
nvidia-smi
ss -tlnp 2>/dev/null | grep -E ":(5432|6379|8000|3000|15432|16379|18000|13000)" || echo "기본 포트들 모두 비어있음"
docker ps --format "table {{.Names}}\t{{.Ports}}\t{{.Status}}" 2>/dev/null || echo "Docker 미설치 또는 권한 이슈"
```
